## Download

````terminal
$ git clone https://github.com/ZeyadTarek1/cookingRecipe.git

# Install

## Install client dependencies (PORT: 3000)

```terminal
$ cd client
$ npm i
````

## Install server dependencies (PORT: 5000)

```terminal
$ npm i
```

## Run-Stack

```terminal
$ npm run dev
```
